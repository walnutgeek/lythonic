# v0.0.11

## New: Namespace Tags (`lythonic.compose.namespace`)

Tag-based querying for namespace nodes with a simple expression language.

- **`NamespaceNode.tags`** — `frozenset[str]` field on every node, set at
  registration time via optional `tags` parameter on `register()` and
  `register_all()`.
- **`Namespace.query(expr)`** — filter nodes with tag expressions supporting
  `&` (AND), `|` (OR), `~` (NOT) with standard precedence. Recursively
  searches all branches.
- **Tag validation** — rejects bare strings (common mistake), operator
  characters, and whitespace in tag names.
- **Config round-trip** — `NamespaceEntryConfig.tags` field for persisting
  tags through `load_namespace()` / `dump_namespace()`.
